/*========= Installation =========*/
1. Upload the 'foxeed-lite` folder to the `/wp-content/themes/` directory

/*========= Activation and Use =========*/
1. Activate the Theme through the 'Themes' menu in WordPress.
2. See Appearance -> Customize to change theme options.
3. See Settings -> Reading -> Choose a Page for the Front page and Page for Posts page from drop-down.


/*========= Theme Features and Documentation =========*/
Setting Home Page
1) Create a Page from Dasboard.
2) Go to "Settings >> Reading" and select the created page as "Front page".

Setting Full width Page ( Page : No Sidebar Template )
1) Create a Page from Dasboard.
2) Select option "Page : No Sidebar Template" from "Page Attributes >> Template" drop-down.

== WP Version ==

It's requires at least version 4.1 to work properly. Please make sure your WP version is at least 3.4.

== Theme Setup ==

Go To Appearance -> Foxeed Options For Theme Options.

*General Settings:*
1) Primary Theme Color
2) Secondry Theme Color
3) Change Favicon

*Header Settings:*
1) Change Logo.
2) Social Profile Links
3) On/Off Persistant(Sticky) Header

*Home Page Settings:*
1) ON/OFF Our Service Section. ( Add "Featured Box - Foxeed Lite" from Widgets->Home Featured Sidebar +Add a Widget )
3) ON/OFF Clients Section. ( To change the client logo images open foxeed-lite/includes/front-clients-section.php file and replace the "<?php echo $imagepath.'client-logo.png'; ?>" with your image url )


*Blog Settings :*
1) Enter Blog Page Title.

*Footer :*
1) Enter Your Copyright Text (You can use HTML for links.)
